Conceptos unidad 1

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar contenido en la web. Utiliza etiquetas para definir elementos como encabezados, párrafos, enlaces, imágenes y otros componentes de una página web.

¿Que es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación y el diseño de un documento HTML. Permite controlar aspectos visuales como colores, fuentes, márgenes, y disposición de los elementos en una página web.

¿Que es el DOM?

El DOM (Document Object Model) es una representación estructurada de un documento HTML. Convierte cada elemento de la página (etiquetas, atributos, texto) en un objeto que puede ser accedido y manipulado mediante lenguajes de programación como JavaScript.

DOM y CSS

A través del DOM también podemos modificar los estilos CSS de los elementos. Por ejemplo, con JavaScript podemos cambiar colores, tamaños, fuentes o incluso añadir nuevas clases dinámicamente para alterar la apariencia de la página en tiempo real.

DOM y eventos

El DOM permite escuchar e interpretar eventos como clics, movimientos del mouse, escritura en un campo de texto o la carga de la página. Estos eventos pueden ejecutar funciones, haciendo la página más interactiva y dinámica.

DOM y animaciones

El DOM también puede controlar animaciones, ya sea cambiando propiedades de estilo en intervalos, usando setInterval, requestAnimationFrame o aplicando y removiendo clases con transiciones CSS. Esto permite crear efectos visuales atractivos como menús desplegables, sliders o animaciones personalizadas.

Unidades de medida estática

En HTML y CSS, las unidades de medida definen tamaños de elementos como anchos, altos, márgenes y tipografías. Se dividen en estáticas y relativas.

Las unidades estáticas (px, pt, cm, mm, in) tienen un valor fijo que no cambia según el dispositivo o pantalla. Son utiles para definir tamaños exactos, pero no son flexibles ni adecuados para diseños responsivos.

Unidades de medida relativas

Las unidades relativas (em, rem, %, vw, vh) se adaptan al contexto del elemento o la pantalla. Son ideales para diseños responsivos, ya que permiten que los elementos escalen según el tamaño de la ventana o la configuración del usuario. Por ejemplo, "em" se basa en el tamaño de fuente del elemento padre, mientras que "rem" se basa en el tamaño de fuente raíz del documento.

Elemento "Property"

En CSS, una propiedad es un atributo que define un aspecto específico del estilo de un elemento HTML. Cada propiedad tiene un nombre y un valor asignado. Por ejemplo, en la regla CSS color: blue;, "color" es la propiedad y "blue" es el valor que se le asigna. Las propiedades controlan aspectos como el color, tamaño, margen, padding, borde, entre otros.

Elemento "media"

La regla @media en CSS permite aplicar estilos específicos según las características del dispositivo o la pantalla, como el ancho, alto, orientación o resolución. Esto es fundamental para crear diseños responsivos que se adapten a diferentes tamaños de pantalla, como móviles, tabletas y escritorios. Por ejemplo, se puede usar @media (max-width: 600px) para aplicar estilos solo en pantallas con un ancho máximo de 600 píxeles.

Elemento "Mask"

Permite controlar la visibilidad de un elemento utilizando imagenes, gradientes o vectores como máscaras para ocultar o revelar partes de su contenido de forma flexible y visual. Esta propiedad es un atajo que agrupa propiedades individuales como mask-image, mask-size, mask-position y mask-repeat para definir el efecto de máscara.

Color animation

La animación de color en CSS permite cambiar gradualmente los colores de los elementos a lo largo del tiempo. Esto se logra utilizando la regla @keyframes para definir los estados inicial y final de la animación, y luego aplicando esa animación a un elemento con la propiedad animation. Por ejemplo, se puede crear una animación que cambie el color de fondo de un botón cuando se pasa el cursor sobre él.

Delay

En CSS, la propiedad animation-delay se utiliza para especificar un retraso antes de que una animación comience a ejecutarse. El valor se define en segundos (s) o milisegundos (ms). Por ejemplo, si se establece animation-delay: 2s;, la animación esperará 2 segundos después de que se active antes de comenzar. Esto es útil para sincronizar animaciones o crear efectos escalonados.

Animation-timing-function

La propiedad animation-timing-function en CSS define la velocidad de una animación a lo largo de su duración. Permite controlar cómo se acelera o desacelera la animación en diferentes puntos del tiempo. Algunos valores comunes incluyen linear (velocidad constante), ease-in (aceleración al inicio), ease-out (desaceleración al final) y ease-in-out (aceleración al inicio y desaceleración al final).

Animation-direction

La propiedad animation-direction en CSS especifica la dirección en la que se ejecuta una animación. Los valores comunes incluyen normal (la animación se reproduce de principio a fin), reverse (la animación se reproduce de fin a principio), alternate (la animación alterna entre normal y reverse en cada ciclo) y alternate-reverse (la animación comienza en reverse y luego alterna). Esta propiedad es útil para crear efectos visuales dinámicos.

Animation-fill-mode

La propiedad animation-fill-mode en CSS determina cómo se aplican los estilos de una animación antes y después de que se ejecute. Los valores comunes incluyen none (los estilos no se aplican fuera de la duración de la animación), forwards (los estilos del último fotograma se mantienen después de que la animación termina), backwards (los estilos del primer fotograma se aplican antes de que la animación comience) y both (combina los efectos de forwards y backwards). Esta propiedad es útil para controlar el estado final de un elemento animado.

Reset

Un "reset" en CSS es un conjunto de reglas que eliminan o normalizan los estilos predeterminados aplicados por los navegadores web a los elementos HTML. Esto asegura que todos los navegadores comiencen con una base consistente, evitando discrepancias en la apariencia de los elementos. Un ejemplo común es el "CSS Reset" de Eric Meyer, que establece márgenes, paddings y otros estilos a cero para todos los elementos.

Mobile first

"Mobile first" es una estrategia de diseño web que prioriza la experiencia del usuario en dispositivos móviles antes de considerar pantallas más grandes como tabletas o escritorios. Esto implica diseñar y desarrollar primero para las limitaciones y características de los dispositivos móviles, y luego escalar hacia arriba para adaptarse a pantallas más grandes utilizando técnicas como media queries en CSS. El enfoque mobile first ayuda a garantizar que los sitios web sean accesibles, rápidos y fáciles de usar en cualquier dispositivo.

Clases separadas por espacios

En HTML, un elemento puede tener múltiples clases asignadas separándolas con espacios en el atributo "class". Esto permite aplicar varios estilos CSS a un mismo elemento, facilitando la reutilización de estilos y la organización del código. Por ejemplo, <div class="clase1 clase2 clase3"></div> aplica los estilos definidos para "clase1", "clase2" y "clase3" al mismo div.

JavaScript

Funciones

Una función en JavaScript es un bloque de código reutilizable que realiza una tarea específica. Se define utilizando la palabra clave function, seguida del nombre de la función y paréntesis que pueden contener parámetros. Las funciones ayudan a organizar el código, mejorar la legibilidad y evitar la repetición. Se pueden invocar (llamar) en cualquier parte del programa para ejecutar el código que contienen.

Funciones que retornan valores

Una función que retorna un valor en JavaScript utiliza la palabra clave return para enviar un resultado de vuelta al lugar donde fue llamada. Esto permite que la función produzca un valor que puede ser almacenado en una variable, utilizado en expresiones o pasado como argumento a otras funciones. Llamada a una función

La llamada a una función en JavaScript se realiza escribiendo el nombre de la función seguido de paréntesis. Si la función requiere parámetros, estos se pasan dentro de los paréntesis. Al llamar a una función, el código dentro de ella se ejecuta y, si la función retorna un valor, este puede ser capturado o utilizado. Por ejemplo, si tenemos una función llamada sumar que toma dos números como parámetros, podemos llamarla así: sumar(5, 3);.

getElementById

El método getElementById en JavaScript se utiliza para acceder a un elemento HTML específico mediante su atributo "id". Este método devuelve una referencia al primer elemento que coincide con el id proporcionado, lo que permite manipular su contenido, estilo o atributos. Por ejemplo, si tenemos un elemento con <div id="miElemento"></div>, podemos acceder a él en JavaScript usando document.getElementById("miElemento").

addEventListener

El método addEventListener en JavaScript se utiliza para registrar un evento en un elemento HTML. Permite especificar una función que se ejecutará cuando ocurra un evento particular, como un clic, movimiento del mouse o pulsación de una tecla. Este método es fundamental para crear interactividad en las páginas web, ya que permite responder a las acciones del usuario de manera dinámica. Por ejemplo, para escuchar un clic en un botón, se puede usar: boton.addEventListener("click", funcion);.
